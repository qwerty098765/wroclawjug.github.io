Wrocław Java User Group (WrocławJUG)

Gdzie nas znaleść:

  • Meetup - tutaj zawsze publikujemy nadchodzące spotkania w pierwszej kolejności, zapisujemy się, aktualizujemy informacje itp.
  • Google Group - tutaj dyskutujemy
  • Twitter - tutaj ćwierkamy, informujemy o spotkaniach itp.
  • Facebook - grupa na Facebooku, działa od niedawna, część Twittów tam ląduje
  • Slack - tutaj też możemy podyskutować. Rejestracja, adress Slacka: jvm-poland.slack.com Kanał: #meetup-wroclaw-jug
  • User Voice - strona na której mieliśmy głosować, o czym chcemy posłuchać / o czym chcemy mówić, raczej zapomniana

Menu:

  • Talks - (niepełna [jeszcze]) lista starych spotkań
